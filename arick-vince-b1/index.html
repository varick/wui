Project Name: Worldsurf

Project Description

The usability test gives us the chance to see how the interface works with the user. Is it easy to navigate? Is it difficult for the user to get done what is expected? Are the buttons to small? Is it frustrating and the person gives up? With this test we can make adjustments to the application make it easy and more streamlined for the user.
